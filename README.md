# 超导材料临界温度预测研究

> 基于高级特征工程的机器学习方法实现高效筛选

## 📋 项目简介

本项目针对超导材料临界温度（Critical Temperature, Tc）的高效筛选与预测问题，提出了一种融合领域知识与机器学习的高级特征工程方法。研究不仅关注预测精度，更注重模型的可解释性，以揭示影响超导临界温度的物理机制。

## 🎯 研究背景

超导材料的临界温度预测是计算材料科学中的核心挑战。传统的实验筛选方法成本高、周期长，而现有的机器学习方法普遍存在以下问题：

- **特征工程同质化**：依赖原始特征，缺乏物理驱动的高级特征
- **可解释性不足**：追求预测精度而忽视模型的物理意义
- **类别不平衡**：超导样本天然稀缺，类别分布不均

## 🔬 研究方法

### 1. 数据预处理

- 异常值修正与缺失值填充
- 特征编码与分箱处理
- 全面统计分析确保数据质量

### 2. 高级特征工程（核心创新）

基于凝聚态物理与材料科学先验知识，设计了多项物理驱动特征：

| 特征类别 | 典型特征 | 物理意义 |
|---------|---------|---------|
| 热力学性质 | 加权平均热导率 | 材料热传导能力 |
| 电子结构 | 加权电子亲和能 | 电子捕获倾向 |
| 价电子统计 | 加权价电子数标准差 | 能带结构复杂度 |

> 核心思路：**不是通过增加特征数量提升精度，而是通过物理驱动特征提升可解释性**

### 3. 类别不平衡处理

- 采用 **SMOTE** 过采样策略
- 在交叉验证框架下实施，确保泛化能力

### 4. 模型训练与评估

- 多种主流模型：LightGBM、XGBoost、CatBoost 等
- **贝叶斯优化**进行高效超参数调参
- **5折交叉验证** + 阈值搜索提升分类精度

### 5. 可解释性分析

引入 **SHAP** 框架从两个层面进行分析：

- **全局层面**：特征重要性排序，揭示主导因素
- **个体层面**：单样本预测解释，理解具体机制

## 📊 实验结果

### 模型性能对比

| 模型 | ROC-AUC | F1-Score |
|------|---------|---------|
| LightGBM | **0.952** | **0.862** |
| XGBoost | 0.948 | 0.851 |
| CatBoost | 0.945 | 0.843 |

### 提升效果

- 融合业务驱动特征的 LightGBM 模型，ROC-AUC 相比基线提升 **0.10**

### 关键发现

SHAP 分析表明，以下三个维度是主导临界温度分类的最关键因素：

1. 🔥 **加权平均热导率** — 热传导演化
2. ⚡ **加权电子亲和能标准差** — 电子得失能力
3. 💎 **加权价电子数标准差** — 能带结构

这验证了超导临界温度由**热、电、量子结构**三大核心维度共同驱动的经典凝聚态物理逻辑。
