# 超导材料临界温度预测研究

> 基于高级特征工程的机器学习方法实现高效筛选

## 📋 项目简介

本项目针对超导材料临界温度（Critical Temperature, Tc）的高效筛选与预测问题，提出了一种融合领域知识与机器学习的高级特征工程方法。研究不仅关注预测精度，更注重模型的可解释性，以揭示影响超导临界温度的物理机制。

## 🎯 研究背景

超导材料的临界温度预测是计算材料科学中的核心挑战。传统的实验筛选方法成本高、周期长，而现有的机器学习方法普遍存在以下问题：

- **特征工程同质化**：依赖原始特征，缺乏物理驱动的高级特征
- **可解释性不足**：追求预测精度而忽视模型的物理意义
- **类别不平衡**：超导样本天然稀缺，类别分布不均

## 🔬 研究方法

### 1. 数据预处理

- 异常值修正与缺失值填充
- 特征编码与分箱处理
- 全面统计分析确保数据质量

### 2. 高级特征工程（核心创新）

基于凝聚态物理与材料科学先验知识，设计了以下物理驱动特征：

#### 2.1 晶格畸变与结构不稳定性

| 特征名称 | 计算公式 | 物理意义 |
|---------|---------|---------|
| 晶格尺寸变异系数 | 加权原子半径标准差 / 加权原子半径均值 | 描述晶格畸变程度，衡量结构不稳定性 |
| 价电子涨落指数 | 加权价电子标准差 / 加权价电子均值 | 反映电子浓度波动，与晶格振动相关 |

#### 2.2 电子结构与成键特性

| 特征名称 | 计算公式 | 物理意义 |
|---------|---------|---------|
| Mulliken 电负性代理 | (加权电离能均值 + 加权电子亲和能均值) / 2 | 表征材料整体电子亲和能力 |
| 绝对化学硬度 | (加权电离能均值 - 加权电子亲和能均值) / 2 | 衡量电子稳定性，与超导配对机制相关 |
| 掺杂主元偏移度 | 加权原子质量均值 - 原子质量算术均值 | 反映元素质量分布的偏移程度 |

#### 2.3 热力学与声子特征

| 特征名称 | 计算公式 | 物理意义 |
|---------|---------|---------|
| 热传导-熔化耦合比 | 加权热导率均值 / 加权熔化热均值 | 描述热传导与晶格稳定性的耦合关系 |
| 摩尔堆积密度 | 加权密度均值 / 加权原子质量均值 | 表征原子堆积紧密度 |
| 质量-声子关联振荡 | 1 / √(加权原子质量均值) | 反映声子频率与原子质量的关联 |

#### 2.4 复杂性与熵效应

| 特征名称 | 计算公式 | 物理意义 |
|---------|---------|---------|
| 绝对构型熵容差 | 元素数量 × 原子质量熵 | 描述元素多样性与质量分布的熵效应 |
| 电离能跨度比 | 电离能范围 / 加权电离能均值 | 反映电子能级分布的跨度 |

> 核心思路：**不是通过增加特征数量提升精度，而是通过物理驱动特征赋予模型可解释的物理意义**

### 3. 类别不平衡处理

- 采用 **SMOTE** 过采样策略
- 在交叉验证框架下实施，确保泛化能力

### 4. 模型训练与评估

- 多种主流模型：LightGBM、XGBoost、CatBoost 等
- **贝叶斯优化**进行高效超参数调参
- **5折交叉验证** + 阈值搜索提升分类精度

### 5. 可解释性分析

引入 **SHAP** 框架从两个层面进行分析：

- **全局层面**：特征重要性排序，揭示主导因素
- **个体层面**：单样本预测解释，理解具体机制

## 📊 实验结果

### 模型性能对比

| 模型 | ROC-AUC | F1-Score |
|------|---------|---------|
| LightGBM | **0.952** | **0.862** |
| XGBoost | 0.948 | 0.851 |
| CatBoost | 0.945 | 0.843 |

### 提升效果

- 融合业务驱动特征的 LightGBM 模型，ROC-AUC 相比基线提升 **0.10**

### 关键发现

SHAP 分析表明，晶格畸变、热力学耦合与电子结构特征是影响超导临界温度的主导因素。这验证了超导临界温度由**结构稳定性、热传导演化、电子特性**三大核心维度共同驱动的经典凝聚态物理逻辑。
